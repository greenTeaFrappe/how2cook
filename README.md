# 🍳How2Cook
## 프로젝트 소개 
How2Cook은 사용자들이 요리 레시피를 검색하고, 공유할 수 있는 프로그램입니다. 사용자들은 자유롭게 자신만의 레시피를 작성하여 공유하거나 찾아볼 수 있는 편리한 기능을 이용할 수 있습니다.

### 기능 소개

**1. 주요 기능**
   - 레시피 검색
   - 레시피 등록, 수정, 삭제

**2. 사용자 기능**
   - 회원등록 
   - 회원탈퇴
